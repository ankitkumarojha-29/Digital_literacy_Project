## Project Overview
This repository contains my Digital Literacy Portfolio, created as part of my assignment as a Student Digital Ambassador. The objective of this project is to build resources that help my peers navigate the digital world safely, professionally, and responsibly. It includes practical applications of digital tools across five distinct modules. All work is organized into this single GitHub repository.

## Module Summaries

* **Task 1: Digital Literacy Awareness Infographic (Module 1)** 
    Created a visual resource using Canva to explain the importance of digital literacy, highlighting useful digital tools and safe internet practices. The final exported design is located in the `task-1-presentation/` folder.

* **Task 2: Student Digital Portfolio (Module 2)** 
    Established a foundational professional online presence by setting up and optimizing profiles on GitHub, LinkedIn, and Kaggle. Screenshots of these profiles are documented in the `task-2-portfolio/` folder.

* **Task 3: Coding & Collaboration Platforms (Module 3)** 
    Explored academic and coding tools by completing a Python programming challenge on HackerRank and designing a 'Digital Literacy Awareness Quiz' using Google Forms and Sheets. Evidence of these activities is saved in the `task-3-platforms/` folder
    * 🔗 **Link to Google Form Quiz:** [https://docs.google.com/forms/d/e/1FAIpQLSe9EcifSpGX_5q0b7DXpAEScf8X31Pen-_qInFPKjbudKt8Lw/viewform?usp=publish-editor] 

* **Task 4: Professional Email & Etiquette Guide (Module 4)** 
    Drafted professional emails for academic (deadline extension) and corporate (internship inquiry) scenarios. Additionally, created a Do's and Don'ts checklist for responsible social media use. These files are located in the `task-4-email-etiquette/` folder.

* **Task 5: Cybercrime Awareness Case Study & Prevention (Module 5)** 
    Researched a realistic cybercrime scenario focusing on UPI Payment Fraud via screen-sharing apps. Developed a case study and an 8-point prevention checklist, including reporting channels like the National Cyber Crime Portal (cybercrime.gov.in) and Helpline 1930. These resources are stored in the `task-5-cybercrime/` folder.
